# Merged Results

Consolidated artifacts from the SBTG functional connectivity pipeline. 

---

## Contents

### Core Result Files

| File | Description |
|------|-------------|
| `result_C_merged.npz` | Multi-lag SBTG results |

### Directories

| Directory | Contents |
|-----------|----------|
| `figures/` | Publication figures and evaluation CSVs (AUROC, AUPRC, F1 tables) |
| `state_dependent/` | Per-method state-dependent analysis (Pearson, Granger, SBTG branches) |
| `state_dependent_analysis/` | Cross-method state comparison and figures |
| `synthetic_sbtg_results/` | Synthetic benchmark outputs (metrics, plots, best parameters, synthetic reports) |
| `external_baselines/` | External DL baseline workspace (NRI, NetFormer, LINT) — see its own README |

### Synthetic Scaling Reports

Synthetic scaling writeups and supporting files live under
`synthetic_sbtg_results/`. Key files:

| File | Contents |
|------|----------|
| `report_complete.md` (in run output dirs) | Full-detail report with raw/filtered tables, success rates, and failure details |
| `baseline_status.csv` (in run output dirs) | Per-method status (`SUCCESS`, `FAILED`, `TIMEOUT`, `SKIPPED`) with errors |
| `method_success_rates.csv` (in run output dirs) | Attempted/successful counts and success rates by method |
| `metrics_common_success.csv` (in run output dirs) | Family-wise common-success subset metrics |
| `sbtg_results_synthetic_extra_run/extra_metrics.csv` | Fast baseline-only synthetic run metrics (`PCMCI+`, `DYNOTEARS`) |
| `sbtg_results_synthetic_extra_run/extra_baseline_status.csv` | Fast run status/error table for `PCMCI+`/`DYNOTEARS` |
| `sbtg_results_synthetic_extra_run/extra_summary.json` | Fast run aggregated metrics by family/method |

## Evaluation CSVs (in `figures/`)

| File | Contents |
|------|----------|
| `eval_cook_C.csv` | AUROC, AUPRC, F1 vs `Cook_Synapses_2019` per lag per method |
| `eval_leifer_C.csv` | Same against `Randi_Optogenetics_2023` |
| `eval_monoamine_all_methods.csv` | Per-transmitter AUROC for all methods and lags |
| `chem_gap_analysis.csv` | Chemical vs gap junction breakdown |
| `baselines_info.json` | Baseline method metadata |

These CSVs are generated by `analysis/evaluation/prepare_merged_results.py`.

---

## Regenerating

To regenerate evaluation CSVs and figures from the `.npz` files:

```bash
python analysis/evaluation/prepare_merged_results.py
python analysis/figures/generate_merged_figures.py
```
